必要なもの(作成時の動作環境)

NIVISA (NIVISA1400)
NI488.2 (NI4882_1760f0)
python (python3.9.7)




__________NIVISAの設定(うろ覚えなので違うかも)_______________

NIVISAのsetup.exeを起動
デフォルトの設定のままインストール

NI4882のsetup.exeを起動しこれもデフォルト設定でインストール

	NIVISAが正常に動作しているかの確認方法:

		スタートメニューの検索バーにNIMAXと打てば多分でてくるので起動する(なければNIVISAのインストールし直し)
		NIMAXの｢デバイスとインターフェース｣の左のプルダウン(∨みたいなやつ)をクリック. 
		GPIBがつながっていればさらにプルダウンがあるアイテムがでてくるのでプルダウンをクリック.
		自分の通信したい機器を選択して｢計測器と通信する｣をクリック.
		送信文字列に"*IDN?"を入力して｢クエリ｣をクリック.
		装置の名前がかえってこれば問題なし


__________pythonの設定(wifi環境必須)________________

pythonなんとか.exeをダブルクリックで起動

Add Python 3.9 To Path みたいなチェックマークにチェック
(チェックをつけずにインストールしてしまったときはもう一度インストーラーを起動すればよい)

その他は特に設定を変えずに進めてインストールを完了させる.

このファイルがあるフォルダで何も選択していない状態で背景の白いところをshiftキー押しながら右クリック

｢PowerShellウィンドウをここで開く｣ を選択

PowerShellで "python -m pip install -r requirements.txt" を入力しエンター

インストールが終わってSuccessfullyなんちゃらってメッセージがでたら完了


(WARNING: You are using pip version~みたいな警告は無視して良い)


	pythonの設定がうまくいっているかの確認:

		実行ファイル(exeファイル,おそらくGPyM.exeというファイル)を実行して
		定義ファイルはGPyM\Example\define.def
		測定マクロはtest.pyを選択
		ちゃんと動いてGPyM\Example\dataにデータが入っていればOK
		(このマクロはGPIB通信を行わないのでNIVISAの確認はできない)
		(GPIB通信の確認をするにはマクロでoperation_checkを選択)
